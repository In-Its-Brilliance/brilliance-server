## Синхронизация чанков

1) Update тик сервера
   - система send_chunks() вызывается каждый тик
   - она сама решает, есть ли что отправлять

2) Внутри send_chunks()
   - смотрит мир и список видимых чанков игрока
   - если чанк не в send_chunk_queue и не в already_sended
   - вызывается send_chunk()
   - чанк добавляется в send_chunk_queue
   - чанк отправляется клиенту по сети

3) Асинхронно по сети
   - клиент получил чанк
   - клиент отправил серверу сообщение
     что этот чанк получен и загружен

4) Network система сервера получает это сообщение
   - вызывается mark_chunks_as_recieved()
   - чанк убирается из send_chunk_queue
   - чанк добавляется в already_sended

5) Игрок сместился между чанками
   - мир вычисляет ChunkChanged
   - вызывается send_chunks_to_unload()
   - чанк убирается из already_sended
